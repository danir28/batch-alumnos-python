# Batch de Alumnos

Proyecto batch simple en Python para gestionar alumnos y sus notas.

Permite:
- Agregar alumnos sin duplicados
- Calcular estadísticas (promedio, aprobados, desaprobados, nota máxima y mínima)
- Mostrar resultados por consola

## Estructura del proyecto

src/
└── batch_alumnos/
    ├── main.py
    ├── core/
    │   └── logic.py
    ├── models/
    │   └── alumnos.py
    └── utils/
        └── validaciones.py

## Cómo ejecutar el proyecto

Desde la carpeta raíz del proyecto:

```bash
python main.py
